Опис предметного середовища

Предметне середовище охоплює розробку простого веб-застосунку “To-Do List”, який дозволяє користувачу створювати, редагувати, позначати виконані завдання та видаляти їх. Дані зберігаються у localStorage браузера.

Структура головної сторінки WEB-застосунку

Головна сторінка містить:

  • Заголовок сторінки та логотип.
  • Форму для додавання нового завдання.
  • Список завдань із кнопками редагування та видалення.
  • Таблицю прикладу.
  • Посилання на сторінку опису застосунку.

HTML-код таблиці

<table>
  <tr><th>№</th><th>Завдання</th><th>Статус</th></tr>
  <tr><td>1</td><td>Підготувати звіт</td><td>Активне</td></tr>
  <tr><td>2</td><td>Придбати продукти</td><td>Виконано</td></tr>
</table>

Скріншот таблиці:

Теорія: Таблиці створюються за допомогою тегів: <table>, <tr>, <th>, <td>. Основні атрибути: border, cellpadding, cellspacing, width, align.

HTML-код зображення

<img src="todo-preview.png" alt="Скрін To-Do застосунку" width="400">

Скріншот зображення:

Теорія: Тег <img> використовується для вставки зображень. Основні атрибути: src — шлях до зображення, alt — альтернативний текст, width / height — розміри, title — підказка при наведенні.

HTML-код списків

<ul>
  <li>Додати завдання</li>
  <li>Позначити виконаним</li>
  <li>Видалити завдання</li>
</ul>

Скріншот списку:

Теорія: Списки створюються тегами: <ul> — маркований, <ol> — нумерований, <li> — елемент списку. Атрибути: type (тип маркера або номера), start (початкове число), reversed (зворотній порядок).
